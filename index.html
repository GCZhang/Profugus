Welcome to Profugus

Profugus is a set of mini-apps culled from the Exnihilo Transport Code system at Oak Ridge National Laboratory. Hence the name Profugus, which is latin for exiled or fugitive/fleeing/escaping, Profugus is the wandering vagrant of Exnihilo. The best way to get started with Profugus is to examine the Development Manual.
